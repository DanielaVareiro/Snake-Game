# 🐍 Classic Snake Game

A fast, clean, and classic arcade Snake Game built entirely with vanilla **HTML5, CSS3, and JavaScript**. 

---

## 🎮 How to Play

1. Use your **Arrow Keys** (Up, Down, Left, Right) to control the snake.
2. Eat the red food to grow and increase your score.
3. **Avoid the walls and your own tail!** If you crash, it's game over.

---

## 🛠️ Tech Stack

* **Structure:** HTML5 Canvas
* **Styling:** CSS3
* **Logic:** Vanilla JavaScript (Game loop, collision detection, and score tracking)
